# Standup Notes

Lightweight template for our daily (or every-other-day) standups. The goal is to keep each round short, surface blockers quickly, and leave a written trail anyone can skim later.

How to use:
- Copy the **Meeting entry** block below to the top of this file for every new standup.
- Each teammate fills their own sub-section under **Team updates** in ≤ 2 minutes of speaking time.
- Note blockers explicitly — if something is blocked, someone owns unblocking it before the next standup.
